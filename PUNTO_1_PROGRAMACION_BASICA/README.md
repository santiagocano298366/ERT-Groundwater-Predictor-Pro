# Punto 1: Programación Básica en Python

## Descripción
Validación de mediciones ERT con condicionales (IF/ELIF/ELSE), ciclos (FOR/WHILE) y manejo de alertas.

## Objetivo
Implementar lógica de programación básica para procesar 15 mediciones de resistencia eléctrica.

## Resultados
- ✅ 15 mediciones procesadas
- ✅ 86.7% operación normal
- ✅ 100% incertidumbre aceptable

## Archivos
- `codigo/punto_1.py` - Código fuente
- `resultados/01_validacion_mediciones.csv` - Validación de mediciones
- `resultados/01_incertidumbre_mediciones.csv` - Análisis de incertidumbre

## Conceptos aplicados
- IF/ELIF/ELSE para validación
- FOR para iterar mediciones
- WHILE para análisis de incertidumbre
- Estructuras de datos: listas y diccionarios
