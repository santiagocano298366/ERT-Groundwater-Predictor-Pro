# Punto 4: Simulación de Monte Carlo

## Descripción
5,000 simulaciones de Monte Carlo para estimar probabilidad de presencia de agua subterránea.

## Objetivo
Cuantificar incertidumbre en mediciones y predicciones.

## Resultados
- ✅ 5,000 simulaciones completadas
- ✅ **81.5% probabilidad de agua**
- ✅ Distribución normal (Gauss) validada
- ✅ Intervalo de confianza 90%: [-0.17, 0.64] S/m
