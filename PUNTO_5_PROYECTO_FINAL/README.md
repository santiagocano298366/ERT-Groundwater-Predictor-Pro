# Punto 5: Proyecto Final + Revisión de Literatura

## Descripción
Avance del proyecto de Dispositivo Portátil de Medición para predicción de agua subterránea.

## Objetivo General
Desarrollar dispositivo automatizado, portátil y accesible para predicción de agua.

## Objetivos Específicos
1. Validación técnica (85-90% precisión)
2. Desarrollo de hardware (<2 kg, <$5,000)
3. Desarrollo de software (API + app mobile)
4. Validación en campo (20+ sitios)
5. Prototipado (5 dispositivos)

## Revisión de Literatura
- ✅ 10 referencias recientes (2020-2025)
- ✅ 70% posteriores a 2022
- ✅ Métodos ERT validados
- ✅ IA alcanza 90% precisión

## Cronograma
- Meses 1-2: Análisis de datos
- Meses 3-4: Modelo ML
- Meses 5-7: Hardware + Software
- Meses 8-10: Validación campo
- Meses 11-12: Prototipado
